# Сборка фронтенда
1. Переходим в папку [frontend/](./)
2. Выполняем установку всех зависимостей
    ```bash
    npm install
    ```

3. Выполняем генерацию api (в этот момент должен быть запущен сервер)
    ```bash
    npm run generate-api
    ```

4. Выполняем сборку фронтенда
    ```bash
    npm run build
    ```
